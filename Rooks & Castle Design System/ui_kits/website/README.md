# Rooks & Castle — Website UI Kit

## Overview
High-fidelity recreation of the AI agency marketing website.
Based on: `rooksandcastle/ai-agency-site` (React/Vite/Tailwind, main branch)

## Screens
- **Home** — Nav + Hero + Services + Process + Stats + Footer
- Click-through: nav links scroll to sections

## Components
- `Navigation` — glass pill nav with dropdowns
- `Hero` — full-bleed dark with green gradient headline
- `ServiceCards` — 3-column grid with icon + glow hover
- `ProcessSteps` — 3-step cards with connectors + stats bar
- `Footer` — newsletter + link columns + social icons

## Design Notes
- Dark mode only (#0F0F0F bg)
- Primary green: #10B981
- Fonts: Inter (body), Playfair Display (display heads), Montserrat (labels)
- Icons: Lucide (outline, strokeWidth 1.5)
- No Framer Motion in kit — CSS transitions only for simplicity
